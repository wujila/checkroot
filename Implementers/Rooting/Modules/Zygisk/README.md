### Zygisk

This folder aims to archive the latest versions of different Zygisk implementations. 

Please also consider using the latest one generated in the ``Actions`` tab of the corresponding GitHub repository if open-source. 

- Magisk built-in Zygisk
- Zygisk Next
  - Versions
    - Zygisk Next v1.3.0 and later: Suitable for almost all the cases with most Shamiko features integrated
    - Zygisk Next v1.2.9.1 and before: Suitable for KSU, Magisk, and their variants with Shamiko
  - Links
    - [https://github.com/Dr-TSNG/ZygiskNext](https://github.com/Dr-TSNG/ZygiskNext)
    - [https://t.me/real5ec1cff](https://t.me/real5ec1cff)
- [NeoZygisk](https://github.com/JingMatrix/NeoZygisk/actions): Suitable for Apatch that does not support Shamiko and be friendly to configure a whitelist superuser tab
- [ReZygisk](https://github.com/PerformanC/ReZygisk/actions): Suitable for KSU and its variants that are adequately hidden and wishes to reduce battery consumption

Compared with others, Magisk built-in Zygisk can be easily detected, Zygisk Next is closed-source, and NeoZygisk uses ``ptrace``.

### Zygisk

本文件夹旨在存储不同 Zygisk 实现的最新版本。

对于开源存储库，请一并考虑使用相应 GitHub 存储库中 ``Actions`` 选项卡里的最新版本。

- 面具内置 Zygisk
- Zygisk
  - 版本
    - Zygisk Next v1.3.0 及以后：适用于绝大多数情况（集成了大部分 Shamiko 功能）
    - Zygisk Next v1.2.9.1 及之前：适用于与 Shamiko 搭配使用的 KSU、Magisk 以及两者的变体的场景
  - 链接
    - [https://github.com/Dr-TSNG/ZygiskNext](https://github.com/Dr-TSNG/ZygiskNext)
    - [https://t.me/real5ec1cff](https://t.me/real5ec1cff)
- [NeoZygisk](https://github.com/JingMatrix/NeoZygisk/actions)：适用于无法与 Shamiko 搭配使用和白名单配置模式的 Apatch 场景
- [ReZygisk](https://github.com/PerformanC/ReZygisk/actions)：适用于隐藏性已足够并希望减少电量开销的 KSU 及其变体的场景

与其它相比，面具内置 Zygisk 很容易被检测到，Zygisk Next 是闭源的，NeoZygisk 使用 ``ptrace``。
