#### Kernel modules

The installation package for a kernel module is usually a ``.kpm`` file. 

Currently, kernel modules can only be installed as embedded modules, and there seem to be very few kernel modules available for search on the internet. 

---

#### 内核模块

内核模块的安装包通常是一个 ``.kpm`` 文件。

目前，内核模块只能以嵌入的形式安装，且在互联网上能搜索到的内核模块似乎不多。
